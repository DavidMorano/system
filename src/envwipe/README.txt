ENVWIPE

This program provides a means to execute another program with a very controlled
environment.  

Synopsis:
$ envwipe [-e <envfile>] <prog> <arg(s)>

Arguments:
[-e <envfile>]	optional environment file
<prog>		program to execute
<arg(s)>	argument(s) to executed program
-V		print program version to standard-error and then exit
