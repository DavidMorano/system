SYSLOGD

This is a knock-off of SYSLOGD (the popular system-logger program).
The difference between this one and the Sun supplied one is that
this one doesn't go into an infinite loop when a bug is triggered! :-)
Suck it down Sun!  Please get your act together.


Building:

$
$ LD_LIBRARY_PATH=
$ export LD_LIBRARY_PATH
$ 
$ make 
$
