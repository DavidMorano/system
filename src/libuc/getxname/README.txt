GETXNAME

These subroutines retrieve various data about the current machine or
system.  The available functions are:
	getchostname
	getclustername
	getfname
	getgroupname
	getnodedomain
	getnodename
	getournetname
	getpjid_name
	getprojname
	getpwusername
	getrealname
	getsysdomain
	getsysname
	getuserdomain
	getusername
	getxusername
	libuc::prgetclustername
