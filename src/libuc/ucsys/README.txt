UCSYS

This directory contains the adaptation code for providing missing
facilities or capabilities normally provided through the standard
UNIX® C-language library (STDC) within the various operating systems
supported.

Current operating systems supported include:

+ SunOS (Solaris)
+ Apple Darwin
+ Linux

The following modules contain the codes for the various system SYSDB
access facilities:

interface-component	system SYSDB facility adapated
------------------------------------------------------

spwd			SHADOW
userattr		USERATTR

ucsyspw			PASSWD
ucsysgr			GROUP
ucsyspj			PROJECT

ucsyspr			PROTOCOLS
ucsysnw			NETWORKS
ucsysho			HOSTS
ucsyssv			SERVICES
