UCGET

This interface group provides for accessing the various system SYSDB
databases.  The various particular databases are accessed through
the following interfce-group components:

component	system database accessed
-------------------------------------------------

ucgetpw		PASSWD
ucgetsp		SHADOW
ucgetua		USERATTR
ucgetgr		GROUP
ucgetpj		PEOJECT

ucgetus		SHELLS

ucgetpr		PROTOCOLS
ucgetnw		NETWORKS
ucgetho		HOSTS
ucgetsv		SERVICES
