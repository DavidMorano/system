UCENT

This directory contains an interface group for operating on system
SYSDB entries.  The following entries have interfaces for entry
management:

file		system SYSDB entry
---------------------------------------------

ucentpw		PASSWD
ucentsp		SHADOW
ucentua		USERATTR
ucentgr		GROUP
ucentpj		PROJECT

ucentpr		PROTOCOLS
ucentnw		NETWORKS
ucentho		HOSTS
ucentsv		SERVICES
