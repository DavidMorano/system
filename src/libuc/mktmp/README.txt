MKTMP

These subroutines make temporary files in the (so-called) "temp"
directory (wherever that may be).

Subroutines included are:

int mktmpdir(char *,cchar *,mode_t) noex
int mktmpusers(char *) noex
int mktmpuser(char *) noex
int mktmpuserx(char *,cchar *) noex
int mktmpuserdir(char *,cchar *,cchar *,mode_t) noex
int mktmpfile(char *,cchar *,mode_t) noex
int mktmplock(char *,cchar *,mode_t) noex
