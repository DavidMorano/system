SHADOWORDER

This program creates a SHADOW file DB that is ordered the same as the PASSWD
file DB.

Synopsis:
$ shadoworder [-dd <dir>] [-p] [-o <opt(s)>] [-V]

Arguments:
-dd <dir>	database directory (default is '/etc')
-p		print out a correctly ordered SHADOW file
-o <opt(s)>	option(s)
-V		print program version to standard error and then exit
