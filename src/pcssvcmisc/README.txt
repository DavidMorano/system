PCSSVCMISC

This directory comtains the source-code subroutines to implement some of the
loadable PCS facility-open-services. Services implemented in this directory are:

service			description
-------------------------------------------------------------------------------

name[­<user>]		connect to the PCS "name" of the user
projinfo[­<user>]	connect to project-information on a user
pcsorg[­<user>]		connect to PCS organization
pcsmotd[­<user>]	connect to PCS Message-Of-The-Day (MOTD)
mailmsgs[­<user>]	connect to PCS mail messages list summarizer
newsdirs		connect to PCS BBNEWS news-drectory list
newsgroups		connect to PCS BBNEWS newsgroup list
