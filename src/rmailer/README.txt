RMAILER

This is the famous, cheapy, RMAILER program. It sends email! This version always
compresses the email message before transmission. This is good on slow links but
may be slow on fast links. Be sure to configure the receiving daemon for
<compressed> messages!

#
# Synopsis:
#	rmailer [recepient(s)] [-r rfile] [-f from] [-M mailer]
#
