TEMP

This is a KSH shell built-in command.  We take temperature specifications as
input (as arguments) and output them in units of both Fahrenheit and Celsius.

Synopsis:
$ temp [<temp(s)>] [-af <afile>] [-V]

Arguments:
<temp(s)>	temperature specifications: in form <v>[c|f|k]
-af <afile>	argument file of <temp(s)>
-V		print command version to standard-error and then exit
